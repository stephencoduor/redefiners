About Web ZIP Export

Web ZIP exports package your course content as a self-contained website that can be viewed offline in any web browser.

Included in export:

  • HTML pages with navigation
  • Embedded images and media
  • Downloadable file attachments
  • Table of contents index

Interactive elements like quizzes and external tools will be exported as static snapshots and cannot be submitted offline.
