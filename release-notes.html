Canvas LMS v2026.03

Released March 15, 2026 | Production Deploy: March 18, 2026

New Features
AI-Powered Learning Assistant

Students now have access to an AI tutor that provides contextual help based on course materials. The assistant can explain concepts, work through practice problems, and suggest study resources.

Enhanced Mastery Paths

Mastery paths now support branching with up to five conditional paths based on quiz score ranges. Instructors can configure custom score thresholds and assign different content sets for each path.

Discussion Analytics Dashboard

New insights panel for discussions showing participation rates, engagement timelines, word clouds, and top contributor rankings. Helps instructors identify students who need engagement support.

Improvements
Gradebook Performance Optimization

Gradebook loading times reduced by 40% for courses with more than 200 students. Pagination and virtual scrolling now enabled by default.

Mobile Responsive Improvements

Assignment submission flow, calendar views, and discussion threads are now fully responsive with improved touch interactions on mobile devices.

Accessibility Enhancements

Improved screen reader support across the platform. All interactive elements now have proper ARIA labels. Keyboard navigation is consistent across all pages.

Bug Fixes

Fixed an issue where assignment due dates were not displaying correctly when crossing daylight saving time boundaries.

Resolved a bug where file uploads exceeding 25MB would silently fail without error notification to the student.

Fixed peer review assignments not appearing in the student To-Do list after being auto-assigned.

Corrected grade calculation rounding errors in weighted assignment groups when using percentage-based grading.

Fixed calendar events not syncing properly with external iCal subscriptions.

Known Issues

SpeedGrader may occasionally display a blank page when switching rapidly between student submissions. A page refresh resolves the issue. Fix targeted for v2026.04.

The new AI assistant is currently available only in English. Multilingual support will be added in a future release.
